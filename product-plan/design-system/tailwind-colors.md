# Tailwind Color Configuration

## Color Choices
- **Primary:** `indigo` — buttons, active states, links, focus rings
- **Secondary:** `slate` — tags, badges, secondary buttons, hover states
- **Neutral:** `zinc` — backgrounds, borders, text, cards

## Usage Examples
Primary button: `bg-indigo-600 hover:bg-indigo-500 text-white`
Active nav item: `bg-indigo-600 text-white`
Secondary badge: `bg-slate-800 text-slate-300`
Card background: `bg-zinc-900 border border-zinc-800`
Muted text: `text-zinc-500`
Subtle border: `border-zinc-800`
