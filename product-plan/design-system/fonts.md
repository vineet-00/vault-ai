# Typography Configuration

## Font Stack
- **Heading:** Geist Variable
- **Body:** Geist Variable
- **Mono:** Geist Mono Variable

## Installation (already done in vault-ai)
```bash
bun add @fontsource-variable/geist
bun add @fontsource-variable/geist-mono
```

## CSS Import
```css
@import "@fontsource-variable/geist";
@import "@fontsource-variable/geist-mono";
```

## Usage
- All UI text: Geist Variable
- Code, paths, shortcuts, timestamps: Geist Mono Variable
